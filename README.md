# คู่มือ jamovi: Feedback & Discussions
### คู่คู่มือใช้งานโปรแกรม Jamovi

ยินดีต้อนรับสู่พื้นที่สำหรับแจ้งข้อเสนอแนะและพูดคุยเกี่ยวกับหนังสือ **คู่มือใช้งานโปรแกรม Jamovi**

## 🛠️ จุดประสงค์ของ Repository นี้
Repository นี้ถูกสร้างขึ้นเพื่อใช้เป็นระบบหลังบ้านสำหรับ **Giscus** (ระบบคอมเมนต์) ที่ปรากฏบนหน้าเว็บไซต์หนังสือ เพื่อให้ผู้อ่านสามารถ:

1.  **แจ้งคำผิด (Report Typos):** หากพบจุดที่สะกดผิด หรือเนื้อหาคลาดเคลื่อน แจ้งได้ทันทีครับ
2.  **ข้อเสนอแนะ (Suggestions):** เสนอแนะเนื้อหาที่อยากให้เพิ่ม หรือส่วนที่อ่านแล้วเข้าใจยาก
3.  **ถาม-ตอบ (Q&A):** แลกเปลี่ยนเทคนิคการใช้ jamovi ในการวิเคราะห์ข้อมูลจริง
4.  **Update แจ้งเตือน:** ติดตามการอัปเดตเนื้อหาตามเวอร์ชันใหม่ของ jamovi
